# 📊 SECI1143 Probability and Statistical Data Analysis
## Personal Reflection 
 
**Course:** SECI1143 Probability and Statistical Data Analysis
**Semester:** 2 (2023/2024) | Section 03
**Lecturer:** Dr Nor Haizan bt Mohamed Radzi

## Overall Takeaway
 
Across these assignments and projects, I came to appreciate how statistics is less about numbers and more about asking the right questions, choosing the right tools, and being honest about the limitations of the data. The combination of primary data collection and secondary dataset analysis gave a well-rounded view of the full research workflow — from survey design all the way to formal hypothesis testing.
 
Using **RStudio** throughout also built my confidence with statistical computing, which I know will be useful in more advanced data engineering and analytics work going forward.
