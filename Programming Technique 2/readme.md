# Self-Reflection 

Working on EasyBus deepened my understanding of object-oriented programming in C++, particularly how inheritance and polymorphism work in practice — having `Admin` and `Passenger` both inherit from a base `User` class with an overridden `menu()` function made the system cleaner and more organised than I expected. Managing data through fixed-size arrays taught me to be mindful of boundaries and manual array shifting, especially when implementing the cancel booking feature, and writing ticket details to a `.txt` file using `ofstream` was my first real experience with file I/O in C++, which made the output feel tangible. If I were to revisit the project, I would replace the fixed arrays with `std::vector` for flexibility, add stronger input validation, and better separate UI logic from business logic — but overall, this project gave me a solid foundation in applying OOP concepts to build a functional, structured system from scratch.
